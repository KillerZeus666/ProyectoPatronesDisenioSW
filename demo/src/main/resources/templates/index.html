Un espacio para gestionar y dar seguimiento a sus quejas y solicitudes

Les damos la bienvenida al portal Onmichannel. A través de este espacio, la ciudadanía podrá radicar de manera respetuosa sus quejas relacionadas a una entidad; así como, hacer seguimiento de las mismas.

Funcionalidades

A continuación, las principales funcionalidades de nuestra aplicación.

Recepción de Quejas

Presente sus quejas a través de correo electrónico, sitio web o aplicación móvil, garantizando accesibilidad y facilidad de uso.

Enrutamiento Inteligente

El sistema dirige cada queja a la empresa correspondiente automáticamente.

Unificación automática

El sistema agrupa quejas provenientes del mismo ciudadano u hogar, evitando duplicidades y optimizando la gestión.

Semáforo de Vencimiento

Monitorea tiempos y alerta sobre quejas próximas a vencer.

Tablero de Control

Visualiza y gestiona quejas en tiempo real con filtros personalizados.

Investigación Incumplimiento

Quejas vencidas se escalan automáticamente para su análisis.
